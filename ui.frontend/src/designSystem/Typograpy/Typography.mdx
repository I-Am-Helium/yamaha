# Typography

Yamaha provides a clear hierarchy and a simple set of systematic and accessible typographic styles to apply structure to your interface.

## Global Styles

We have applied some global styles to typography and link texts. This includes the following:

- The body font family is `Ubuntu`
- The base font size is `16px`
